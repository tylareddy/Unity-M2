# Unity-M2

Title: Elden Ring Pin ball
Genre: Physics-based arcade game

A pinball game in the style and theme of Elden Ring with the targets being Main and mini bosses in the game and with a limited ammount of lives/pin balls.and on the background there will be smaller enemies that can give smaller ammount of score and or multipliers.

Gameplay core
Ball: different weapons from the souls games
Targets or bumps: Different environmental areas from each game like a poison swamp
Score: Depending on the boss it will be points between 1-5
Objective: defeat every boss and mini boss styles ball
